# Phone app

This is the phone app made by Joaquin Carbonell.

## Installation

Use the npm to install depencecies

```bash
npm install
```

## Usage

Open one terminal and run the server on port 8000

```pyhton
npm run dev
```
then open a new tab on the terminal and run

```pyhton
npm start
```

## Tests

To run simple tests

```pyhton
npm test
```
